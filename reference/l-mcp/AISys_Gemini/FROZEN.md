\# Frozen Reference Implementation (L-MCP)



This directory contains a \*\*frozen reference implementation\*\* of the

MCP-OSS architecture (L-MCP).



\- Status: Frozen

\- Scope: Reference only

\- Normative source: RFC-0000

\- Changes: Not accepted



This implementation is intentionally preserved in its validated state.

Any evolution of the architecture must occur via new RFCs and separate

implementations.
